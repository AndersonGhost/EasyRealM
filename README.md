# EasyRealM
基于RealM转发工具制作的一键脚本，测试环境debian9，其他没试过。
![image](https://mxpic.ml/2021/01/23/20210123_704052f0f88df.png)
## 使用方法
一键脚本：
`wget -N --no-check-certificate https://github.com/seal0207/EasyRealM/blob/main/realm.sh && chmod +x realm && ./realm.sh`
